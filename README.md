# Folium 라이브러리로 경북 산사태 대피소 GIS 분석하기(DSJA 3기 개인 프로젝트)

안녕하세요! 경상북도 안전을 한눈에 파악할 수 있는 `dsja-shelter-map` 프로젝트에 오신 것을 환영합니다! 🎉

## 왜 이 프로젝트를 만들었을까요? 🤔

안전은 언제나 중요하지만, 비상 상황이 발생했을 때 우리 동네 대피소가 어디 있는지 정확히 아는 사람은 많지 않을 거예요. 이 프로젝트는 **SBS 데이터사이언스와 저널리즘 아카데미 3기** 과정의 일환으로, 데이터를 활용하여 대피소 정보를 시각적으로, 그리고 직관적으로 제공하고자 시작되었습니다.

데이터를 단순히 숫자나 표로 보는 것을 넘어, 지도 위에 생생하게 구현하여 누구든지 우리 주변의 안전시설을 쉽게 파악하고 비상시 빠르게 대응할 수 있도록 돕는 것이 이 프로젝트의 핵심 목표입니다! 💖

## 주요 기능 살펴보기! 👀

이 프로젝트를 통해 다음과 같은 멋진 기능들을 경험할 수 있습니다:

*   **📍 대피소 위치 한눈에 보기:** 지도 위에 현재 위치와 가장 가까운 대피소, 그리고 주변 모든 대피소들을 직관적으로 표시합니다. 이제 비상 상황에도 헤매지 마세요!
*   **🗺️ 다양한 지도 데이터 탐색:** 여러 종류의 대피소 정보(예: 지진 대피소, 민방위 대피소 등)를 필터링하거나, 필요한 상세 정보를 클릭 한 번으로 확인하세요. (예: 수용 인원, 주소, 연락처 등)
*   **📊 데이터 기반 인사이트:** 단순한 위치 정보를 넘어, 데이터 분석을 통해 얻은 유의미한 인사이트를 함께 제공하여 더욱 깊이 있는 이해를 돕습니다. (예: 지역별 대피소 밀집도, 접근성 분석 등)
*   **✨ 사용자 친화적인 인터페이스:** 복잡한 과정 없이 누구나 쉽게 지도를 탐색하고 정보를 얻을 수 있도록 깔끔하고 직관적인 UI를 제공합니다.

### 프로젝트 구성 파일 📁

*   `index.html` / `new_sltr_map.html` : 메인 지도 페이지
*   `basic_map.html` : 기본적인 지도 구현 예시
*   `data/` : 지도에 사용되는 대피소 정보 데이터 파일들
*   `notebook/` : 데이터 전처리 및 분석에 사용된 Jupyter Notebook 파일들 (데이터 분석 과정이 궁금하다면 살펴보세요! 🧑‍💻)
